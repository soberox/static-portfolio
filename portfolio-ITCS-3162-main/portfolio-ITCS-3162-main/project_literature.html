Wherever society arose, storytelling and information served to reflect its beliefs and culture. International best sellers:

I picked this dataset to get a contemporary census on the popularity of literature worldwide. This is not a measure of writing quality or innovative techniques shown by region; this dataset shows which books are being read, and by whom they are written.

Using the contemporary dataset, I created a new dataset from the original one that includes only these relevant categories: rank, date, country, title, author, gender, and nationality. The most interesting column is rank, since it shows a books placement on a countrys bestseller list. To ensure consistency, I dropped every entry with a missing value (NA) for rank.

The date formatting in this dataset was in mm/dd/year, while my other datasets use only the year; therefore, I reformatted the date column to include only the year so it would match the others. Since the other datasets also include country codes, I added a code column based on the listed countries.

I also created a function to format gender consistently as either male or female. Overall, this section took the longest to clean, but the effort was worth it!

#found this method to convert country names to its country code, to format the same as other datasets
import pycountry
def get_country_code(name):
    try:
        return pycountry.countries.lookup(name).alpha_2
    except LookupError:
        return None
df_international['code'] = df_international['country'].apply(get_country_code)

#I also want the gender formatting to be the same as other datasets
def format_gender(gen):
    try:
        if gen == "w":
            return "female"
        elif gen == "m":
            return "male"
        formated = []
        multi = str(gen).split("; ")
        for g in multi:
            if g == "w":
                formated.append("female")
            elif g == "m":
                formated.append("male")
            else:
                formated.append(g)
        string = "; ".join(formated)
        return string
    except TypeError:
        return None 
df_international["gender"] = df_international["gender"].apply(format_gender)

rankings_by_year = (
    best_sellers.groupby(["code", "date"])["rank"]

    .reset_index()

mean_ranks = rankings_by_year.groupby("code")["rank"].mean()
top_codes = mean_ranks[mean_ranks <= 5].index
filtered = rankings_by_year[rankings_by_year["code"].isin(top_codes)]
pivot_rankings = filtered.pivot(index="date", columns="code", values="rank")
plt.figure(figsize=(12, 7))

Conclusion

Through this project, I set out to understand which regions dominate the global literary landscape today and how that compares to historically significant works. By cleaning and analyzing two complementary datasets, contemporary bestsellers and long-term sales/ratings, I was able to highlight both the strengths and limitations of global representation in literature. Ultimately, the findings reveal not only which regions are most prominent, but also how market forces, cultural history, and institutional biases shape the stories that reach readers worldwide. While this study cannot capture the full diversity of global literary traditions, it underscores the importance of broadening our data sources and perspectives if we hope to gain a more balanced view of world literature.
